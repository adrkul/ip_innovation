# Writing Style

**Applies to:** All prose drafting and editing in `Paper/**`, `Theory/Sections/**`, `Slides/**`

When drafting or editing prose for the user, follow these conventions:

- **Voice**: First-person active voice. No passive. No contractions.
- **Tone**: Confident and direct. No unnecessary hedging when claims are supported; hedge only where genuinely uncertain.
- **Sentences**: Build complex ideas through well-structured clauses. Never convoluted. Use parallelism for enumerated points. Avoid breaking sentences with colons, em-dashes, or semi-colons; prefer two separate sentences with a suitable transition.
- **Paragraphs**: Strong topic sentence; each paragraph develops one idea fully.
- **Diction**: Precise and economical. No filler, no purple prose, no empty transitions. Technical vocabulary used naturally but not for its own sake.
- **Limitations**: State assumptions and weaknesses directly, without being defensive.
- **Literature**: Weave citations into the argument rather than siloing them. Situate each contribution relative to what exists.
- **Footnotes**: Use for institutional detail and caveats that would interrupt the main argument.
